# My Factorio/Pyanodons tools.

Some tools for the most complicated processes in the Pyanodons mod
for factorio.

See [`py_neutron_capture.ipynb`](py_neutron_capture.ipnb) for calculation of the plutinium processing chain.

# Setup

Install the following dependencies:

* `uv`
* `pre-commit`
* `ruff`
* `pytest`

Synchronize packages with uv:

```
    uv sync
    uv sync --dev
```
